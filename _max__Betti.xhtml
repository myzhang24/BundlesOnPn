BundlesOnPn :: maxBetti

maxBetti -- produces the maximal Betti numbers

Synopsis

  • Usage:
    (a,b) = maxBetti(n,d,B)
  • Inputs:
    • n, an integer, dimension of the ambient projective space
    • d, an integer, regularity bound
    • B, a list, bundle sequence
  • Outputs:
    • a, a list, degrees of syzygies
    • b, a list, degrees of generators

Description

This function returns the maximal Betti numbers among all normalized bundles on Pn with a given bundle sequence up to a given regularity.

i1 : maxBetti(3,4,{6,4})

o1 = ({2, 2, 2, 3, 4}, {1, 1, 1, 1, 1, 1, 2, 3, 4})

o1 : Sequence

See also

  • minBetti -- produces the minimal normalized Betti numbers of a bundle sequence

  • maxBetti(ZZ,ZZ,List)
