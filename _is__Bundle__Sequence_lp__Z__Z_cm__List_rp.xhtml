BundlesOnPn :: isBundleSequence(ZZ,List)

isBundleSequence(ZZ,List) -- checks whether a sequence is a bundle sequence

Synopsis

  • Usage:
    b = isBundleSequence(n,L)
  • Function: isBundleSequence (missing documentation)
  • Inputs:
    • n, an integer, positive
    • L, a list, of integers
  • Outputs:
    • b, a Boolean value

Description

Checks whether a list of integers is a bundle sequence on Pn.

i1 : isBundleSequence(3,{6,3})

o1 = true
i2 : isBundleSequence(4,{6,3})

o2 = false
