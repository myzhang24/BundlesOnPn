BundlesOnPn :: isAdmissible

isAdmissible -- checks whether a pair of integer sequences is admissible

Synopsis

  • Usage:
    b = isAdmissible(n,A,B)
  • Inputs:
    • n, an integer, dimension of ambient projective space
    • A, a list, the degrees of syzygies
    • B, a list, the degrees of generators
  • Outputs:
    • b, a Boolean value

Description

This functions checks whether a pair of integer sequences (A,B) is admissible on Pn.

i1 : isAdmissible(2,{1,1},{0,0,0,0})

o1 = true

Ways to use isAdmissible :

  • isAdmissible(ZZ,List,List)
