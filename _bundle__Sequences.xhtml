BundlesOnPn :: bundleSequences

bundleSequences -- produces bundle sequences to a given degree

Synopsis

  • Usage:
    L = bundleSequence(n,r,d)
  • Inputs:
    • n, an integer, dimension of ambient projective space
    • r, an integer, rank of the bundle sequences
    • d, an integer, degree bound of the bundle sequences
  • Outputs:
    • L, a list, of bundle sequences on Pn of rank r and degree <= d

Description

This function generates all bundle sequences on Pn of rank r and degree <= d recursively.

i1 : bundleSequences(3,4,9)

o1 = {{4}, {1, 4}, {1, 1, 4}, {1, 1, 1, 4}, {1, 1, 1, 1, 4}, {1, 1, 1, 1, 1,
     ------------------------------------------------------------------------
     4}, {1, 1, 1, 2, 4}, {1, 1, 2, 4}, {1, 1, 3, 4}, {1, 2, 4}, {1, 2, 2,
     ------------------------------------------------------------------------
     4}, {1, 3, 4}, {2, 4}, {2, 2, 4}, {2, 3, 4}, {3, 4}, {5, 4}}
