BundlesOnPn :: minBetti

minBetti -- produces the minimal normalized Betti numbers of a bundle sequence

Synopsis

  • Usage:
    (a,b) = minBetti B
  • Inputs:
    • B, a list, bundle sequence
  • Outputs:
    • a, a list, degrees of syzygies
    • b, a list, degrees of generators

Description

This function returns the minimal Betti numbers among all normalized bundles with a given bundle sequence.

i1 : minBetti {6,4}

o1 = ({2, 2}, {1, 1, 1, 1, 1, 1})

o1 : Sequence

See also

  • maxBetti -- produces the maximal Betti numbers
