BundlesOnPn :: BundlesOnPn

BundlesOnPn -- Bundles On Pn with vanishing lower cohomologies

Description

Bundles On Pn with vanishing lower cohomologies and methods of generating them in a systematic way were described in the paper

Bundles On Pn with vanishing lower cohomologies, https://math.berkeley.edu/~myzhang/research/BundlesOnPnWithVanishingLowerCohomologies.pdf .

This package includes the codes to produce bundle sequences up to a given degree. It also provides methods to generate the maximal and minimal element of the lattice of Betti numbers described in the above paper.

Once the Betti numbers (a,b) are give, we can produce a minimal matrix of random forms in corresponding degrees that gives a presentation of a bundle with vanishing lower cohomologies on Pn.

i1 : R = (ZZ/101)[x,y,z,w]

o1 = R

o1 : PolynomialRing
i2 : L = bundleSequences(3,3,7)

o2 = {{3}, {1, 3}, {1, 1, 3}, {1, 1, 1, 3}, {1, 1, 1, 1, 3}, {1, 1, 2, 3},
     ------------------------------------------------------------------------
     {1, 2, 3}, {2, 3}, {2, 2, 3}, {4, 3}}

o2 : List
i3 : (a,b) = minBetti last L

o3 = ({2}, {1, 1, 1, 1})

o3 : Sequence
i4 : E = coker random(R,transpose matrix apply(#a,i->apply(#b,j-> a#i-b#j)))

o4 = cokernel | 24x-36y-30z-29w  |
              | 19x+19y-10z-29w  |
              | -8x-22y-29z-24w  |
              | -38x-16y+39z+21w |

                            4
o4 : R-module, quotient of R
