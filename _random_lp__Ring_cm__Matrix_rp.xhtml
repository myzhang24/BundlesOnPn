BundlesOnPn :: random(Ring,Matrix)

random(Ring,Matrix) -- produces a minimal matrix of random forms

Synopsis

  • Usage:
    N = random(R,M)
  • Function: random
  • Inputs:
    • R, a ring
    • M, a matrix, of integers specifying the degrees of the random forms of R
  • Optional inputs:
    • Density => ...,
    • Height => ...,
    • MaximalRank => ...,
    • UpperTriangular => ...,
  • Outputs:
    • N, a matrix, of random forms in R of degrees specified by M

Description

This function produces a matrix of random forms in a ring R whose degrees are given by a matrix of integers

i1 : random(ZZ[x,y],matrix{{1,1,2},{2,1,1}})

o1 = | 8x+y        3x+7y 8x2+3xy+3y2 |
     | 7x2+8xy+8y2 5x+7y 8x+5y       |

                      2                3
o1 : Matrix (ZZ[x, y])  <--- (ZZ[x, y])
